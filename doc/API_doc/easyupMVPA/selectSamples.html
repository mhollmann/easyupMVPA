selectSamples

PURPOSE

Select a sub-dataset out of a dataset according to given criteria.

SYNOPSIS

function [dataset] = selectSamples(dataset, criterionString)

DESCRIPTION 

 Select a sub-dataset out of a dataset according to given criteria.

 Author: Maurice Hollmann
 Date  : 08/10

 Description:
 This methods selects samples from the dataset. The selection criterion 
 is given as a string and refers allways to index, chunks or classIDs:

  ds = selectSamples(ds, 'index <= 500')
  ds = selectSamples(ds, 'chunks > 0')
  ds = selectSamples(ds, 'chunks > 0 & classIDs < 2')
  ds = selectSamples(ds, 'chunks > 0 | classIDs < 2')
  ds = selectSamples(ds, 'chunks == 1')

 Parameters:
   dataset          - the datset to set the classIDs for
   criterionString  - selection criterion that will be evaluated as given (i.e. 'index <= 500' or 'dataset.chunks >= 30')

 Returns:
   dataset   - the datset with excluded samples

 Comments:
